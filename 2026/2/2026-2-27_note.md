### mcp流程，通常使用inspector进行调试

```bash
# 模拟client对server请求，同时支持stdio与http协议
npx @modelcontextprotocol/inspector
```

```json
// client请求mcp server初始化，获取mcp-session-id
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "initialize",
  "params": {
    "protocolVersion": "2025-06-18",
    "capabilities": {
      "elicitation": {}
    },
    "clientInfo": {
      "name": "example-client",
      "version": "1.0.0"
    }
  }
}
// 随后发送工具发现（原语），host记录server有哪些工具可用(部分server端会自动推送，也可以client主动更新)
// resources与prompts原语同理，但是这俩不常用
{
  "jsonrpc": "2.0",
  "id": 2,
  "method": "tools/list"
}
// 调用工具，根据host功能与参数记录，ai选择调用相关功能
{
  "jsonrpc": "2.0",
  "id": 3,
  "method": "tools/call",
  "params": {
    "name": "get_file_contents",
    "arguments": {
      "owner": "xx-blbl-xx",
      "path": "/2026/2/2026-2-11_note.md",
      "ref": "refs/heads/master",
      "repo": "notes"
    }
  }
}
```
