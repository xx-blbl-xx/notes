### 现代python项目新建时需要建立一个虚拟环境，隔离各种版本依赖

```bash
# 进入项目目录
cd my_project
# 新建虚拟环境
python3 -m venv .venv
# 进入虚拟环境
source .venv/bin/activate

# 查看pip环境（重点看Location是否为本项目目录）
pip show requests
# 安装各种依赖
pip install -r requirements.txt
```

### __pycache__目录是python自动生成的文件夹，用于存储已编译的 .pyc 字节码文件。它能跳过源代码重新编译，显著加速下次程序启动和模块导入速度。通常在.gitignore中被忽略

### 三种种ai学派

* 符号主义（Symbolism）：常被称为传统人工智能，主张人工智能源于数理逻辑，即通过逻辑符号来表达思维的形成（就是用函数表达一切）。例如各种专家系统，其推理与决策过程通常可追溯，常用于金融医疗等高风险领域。
* 联结主义（Connectionism）：强调仿人脑模型，即将神经元之间的联结关系作为人工神经网络的基础（变量太多，函数表达不了只能将模型简化求近似解）。神经网络和深度学习是其代表。
* 行为主义（Behaviorism）：是融合上述两大范式的优点，注重应用和身体模拟，认为控制论和感知-动作型控制系统是人工智能的关键。例如通用人工智能AGI。

### 在使用大模型接口时，可以通过控制Temperature来控制模型回答是更精准与确定（0-0.3），还是平衡与自然（0.3-0.7），或者更创新与发散（0.7-1.5），非常随机（1.5-2）。由于该参数是解码阶段的超参数，直接控制每次选择下一个词时的概率分布平滑度，所以prompt无法反抗
